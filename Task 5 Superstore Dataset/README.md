# Pandas Superstore Project

## Overview
This project applies Pandas to the Superstore retail dataset (~10,000 US orders) across 8 structured tasks — from data loading to visualization and business insights.

## Files
- `Superstore_Pandas_Project.ipynb` — main notebook with all code, outputs, and commentary
- `Sample-Superstore.csv` — dataset (place in same folder as notebook)


## Libraries used
- `pandas` — data loading, cleaning, groupby, pivot tables, merge
- `matplotlib` — all visualizations

## Key findings
| Question | Answer |
|---|---|
| Most profitable category | Technology |
| Least profitable region | Central |
| Sales vs Profit correlation | ~0.48 (moderate positive) |
| Peak sales period | November–December (Q4 every year) |
| Worst sub-category | Tables (net loss) |
